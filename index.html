Enviornment Setup

Expo

  • A workflow with a GUI, as well as a CLI to create and run your app
  • includes a bunch of APIs, ex: geolocation
  • includes it's own set of developer tools
  • makes it easier to publish your app to iTunes and Google Play
  • can open up an iOs simulator straight from terminal, but not an android emulator
  • requires expo to be downloaded to the device you are testing

React-Native cli

  • Creates a "pure" RN App, since all of the native code is available to the dev
  • this allows the dev to add native modules
  • creates a smaller app rather than having all the extras that come with expo
  • you can run the app directly in Android Studio/Xcode since the native code is exposed

Markup

  • View === Div
  • Text === Paragraph
  • Button/TouchableOpacity === Button/Link
  • Image === Image
  • TextInput === Input/Textarea
  • UL, Radio

Styles

  • Inline styles via style property
  • Class-like styles
  • Properties are camelcased: margin-right marginRight
  • Properties do not allow multiple arguments: margin: 0 10;
  • No Units: margin: 10
  • Flexbox

iOs & Android

  • Some components are only compatible with one system
  • Some components are styled out of the box per their system
  • Some components will require extra setup within their own system
